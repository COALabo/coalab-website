## Overview

Investigating how semantic representations vary across languages using large-scale corpus analysis and computational models. This project aims to understand the universal and language-specific aspects of meaning.

**Lead Researcher:** Dr. Maria Chen  
**Timeline:** 2022-2025  
**Funding:** NIH Grant #R01HD105346

### Key Components

- Multilingual semantic space modeling
- Cross-linguistic metaphor analysis
- Semantic change detection across languages

### Methods

- Distributional semantic models
- Neural network embeddings
- Cross-lingual alignment techniques
